##  Features

- **Welcome new members** with customizable messages and GIFs
- **Anti-link protection** to keep groups clean
- **Remote command system** from GitHub
- **Auto-updating** configuration
- **Message statistics** tracking
- **Command cooldowns**
- **Admin-only commands**
